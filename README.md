# media-queries

erste schritte in css mit media queries

## Einführung

```css
@media (Bedingung){
    /*CSS Code hier, der interpretiert
    wird wenn Bedingung erfüllt ist*/
}
```

### Typische Bedingungen

|Bedingung|Bedeutung|
|---|---|
|'max-width'|Stile gelten bis zu dieser Größe|
|'min-width'|Stile gelten ab dieser Größe|
